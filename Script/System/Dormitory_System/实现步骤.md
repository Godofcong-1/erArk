## 大步骤 5：睡眠恢复效率加成（结算层）

- [x] **5.1** 在宿舍管理系统逻辑文件（`dormitory_manager_system.py`）中新增辅助函数 `get_dormitory_manager_knowledge(character_id: int) -> int`：
  - 根据 `character_data.dormitory` 提取所属层号。
  - 查询 `cache.rhodes_island.dormitory_managers[layer]` 获取该层管理员 id。
  - 返回管理员的学识能力等级（`ability[学识ID]`）；无管理员则返回 0。
- [x] **5.2** 在 `Script/Settle/realtime_settle.py` 的 `settle_sleep()` 中，在"非监禁睡眠"分支内叠加管理员加成：
  - 调用 `get_dormitory_manager_knowledge(character_id)` 获取学识等级。
  - 按公式 `final_adjust *= (1 + knowledge_level * 0.05)` 叠乘。
- [ ] **5.3** 验证：
  - 无管理员时恢复数值与原逻辑一致。
  - 任命管理员（学识 4 级）后，同层角色睡眠恢复提升约 20%。
  - 监禁中角色不受管理员加成影响。
